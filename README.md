# 📊 Retail Sales Analysis

## 📌 Overview
This project analyzes retail sales data using R to generate business insights.

## 🔧 Features
- Revenue calculation
- Best selling product identification
- Product categorization
- High revenue filtering

## 🛠️ Tech Stack
- R
- RStudio

## ▶️ How to Run
1. Open RStudio  
2. Run `sales_analysis.R`

## 📈 Output
![Output](output.png)

This output shows total revenue, average price, and best selling product.
